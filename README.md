# 🚀 Todo & Board Mini Project (Next.js)

## 📝 프로젝트 소개
Next.js 14 (App Router) 환경에서 기본적인 웹 애플리케이션 기능을 구현한 미니 프로젝트입니다. React 컴포넌트와 Next.js의 라우팅 기능을 연습하는 데 중점을 두었습니다.

## ✨ 주요 기능
* **Homepage:** Todo List와 Table(게시판) 페이지로 이동하는 메인 화면
* **Todo List:** 간단한 목록 관리 기능
* **Table (게시판):** 기본적인 게시판 형태의 데이터 테이블 구현
* **페이지 이동:** `next/navigation`의 `useRouter`를 사용한 페이지 간 네비게이션

## 🛠️ 기술 스택
* **프레임워크:** Next.js 14 (App Router)
* **언어/라이브러리:** React
